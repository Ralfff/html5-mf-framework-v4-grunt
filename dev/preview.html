NOTICE Each grid-row is wrapped iside a section. Inside is a .frame-Element which is centered and has a max-width (this represents the content-area). The grid element (e.g. .half) has no padding, so we put a .module-class, which defines our global padding. Everything uses box-sizing: border-box;
